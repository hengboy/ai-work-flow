# 三分类标签

技能以五个标准的三分类角色为依据。本文件将这些角色映射到此仓库问题跟踪器中实际使用的标签字符串。

| AI 工作流技能分布中的标签 | 我们的跟踪器中的标签 | 含义                       |
| ------------------------- | -------------------- | -------------------------- |
| `needs-triage`            | `needs-triage`       | 维护者需要评估此问题       |
| `needs-info`              | `needs-info`         | 等待报告者提供更多信息     |
| `ready-for-agent`         | `ready-for-agent`    | 完全明确，可供离线代理处理 |
| `ready-for-human`         | `ready-for-human`    | 需要人工实现               |
| `wontfix`                 | `wontfix`            | 不会采取行动               |

当某项技能提到某个角色时（例如“应用‘可由代理处理’的三分类标签”），请使用本表中的对应标签字符串。
