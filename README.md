# AI 工作流技能

用于讨论需求、规划本地实施工作、将其分解为依赖感知的任务，并执行相应工作的可移植技能。

## 工作流

`/grill-with-docs` -> `/write-plan` -> `/to-tasks` -> `/implement`

计划和任务存储在目标项目下的 `.ai-work-flow/plans/` 中。技能仓库本身从不是有效的工件目标。

### 生产力类

- `grill-me`
- `grilling`
- `handoff`
- `teach`
- `writing-great-skills`

每个技能目录都包含 `SKILL.md` 和 `agents/openai.yaml` 文件。请将这些目录安装到使用技能的 AI 代理环境中。
